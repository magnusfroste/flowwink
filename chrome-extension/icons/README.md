# Extension Icons

Place PNG icons in this folder:

- `icon16.png` — 16x16 pixels (toolbar)
- `icon48.png` — 48x48 pixels (extension management)
- `icon128.png` — 128x128 pixels (Chrome Web Store)

## Quick Icon Generation

You can use any of these methods:

### 1. Online Tools
- [Favicon Generator](https://favicon.io/)
- [Icon Generator](https://www.icoconverter.com/)

### 2. Design Tools
- Figma, Sketch, or Photoshop
- Export at 16x16, 48x48, and 128x128

### 3. Emoji to Icon
- Use ⚡ emoji as base
- Screenshot and resize
- Or use [Emojipedia](https://emojipedia.org/)

## Temporary Placeholder

Until you add custom icons, the extension will show Chrome's default icon.
The extension will still work perfectly - icons are just for branding.
